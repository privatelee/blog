1.HTML5的介绍

1.1 什么是HTML5?

相信各位看过之前的入门系列，对HTML有了大致了解，那么这里我在做进一步说明。

HTML5是新一代的HTML，是HTML中的最新版本，目前是前端的前沿技术。它由3部分组成，HTML,JavaScript，CSS。

目前支持Html5的浏览器有：Firefox（火狐浏览器）·IE9及其更高版本·Chrome（谷歌浏览器）·Safari·Opera（手机端）·Maxthon（傲游浏览器）·360浏览器·UC浏览器

1.2 HTML5的特性

HTML5的出现产生了新大陆，具体来说就是新元素(新标签)，新属性，完全支持 CSS3，视频和音频，2D/3D 制图，本地存储（cookie=key,value）， 地理定位， Web应用。而我们完全可以用HTML5进行开发微信应用，游戏，视频技术，浏览器技术等。

1.3 HTML5兼容问题

有人说，不是H5的诞生就是为了解决兼容问题么？是的，现代的主流浏览器都支持 HTML5，Chrome支持最好。但是总有人使用非主流（IE6,7,8），而H5对他们是不支持的。低版本浏览器对HTML5的新标签支持不好，但都对无法识别的元素默认作为类似的行内元素处理。
HTML5 定了 8 个新的 HTML 语义（semantic） 元素。所有这些元素都是 块级元素。
